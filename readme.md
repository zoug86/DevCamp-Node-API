# DevCamper API

> Backend API for DecXCamper applicaiton, which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the vaues/setting to your own

## Install Dependicies
```
npm install
```

## Run App
```
# Run in dev mode
npm rn dev

# RUn in prod mode
npm start
```
